## AutoBot v0.1
> Built with **AnnexJS Vanilla r29**  
  
### Features available
> 1. Auto Role new members when they join!  
> 2. Greet new users when they join!  
> 3. Alert you when people leave your server!  
  
### Information
This is a very simple bot that has 3 main jobs.  
With only 2 events!
